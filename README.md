## Project Overview
This is a Cypress-based testing project for Dashclicks.

## Prerequisites
- Node.js (v14.0.0 or later)
- npm (v6.0.0 or later)

## Setup Instructions
1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```

## Running Tests
- To run tests in headless mode:
  ```
  npx cypress run
  ```
- To open Cypress Test Runner:
  ```
  npx cypress open
  ```

## Troubleshooting
- Ensure you have the latest version of Node.js and npm
- If you encounter any dependency issues, try:
  ```
  npm ci
  ```
